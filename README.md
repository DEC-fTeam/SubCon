# 作業フロー

1. GitHub issueを建てる、もしくはAssignする

2. featureブランチを作成する

```bash
git checkout -b feature/機能名

or

git branch feature/機能名
git switch 作成したブランチ名
```

3.

作業する

4. pushする

-uはいらんかも

```bash
git push -u origin 作成したブランチ名
```

5. pull requestsを出す

6. git pullする

developに移動する

```bash
git checkout develop

or

git switch develop
```

pullする

```bash
git pull
```
